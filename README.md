## 基于vue和iview的后台管理系统，侧重于权限控制

> 技术：vue, vuex, iview, webpack, mongodb, node.js

非正式项目，闲暇时间写的一套通用的权限管理模块，便于以后在其他项目中使用。前端利用vue构建，vuex存储状态，iview作为UI框架，webpack打包编译。由于是和后端的同事一起写的，所以这里自己写node只是为自己测试数据方便。
